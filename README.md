# Programação Linear - TP1

## Crie diretorio bin
```
mkdir bin
```
## Compile
```
make
```
## Execute no formato a seguir
```
./simples <arquivo>
```
## Para observar o tableau e outras variaveis pode-se utilizar
```
make TRACE=DEBUG
```
## Para limpar
```
make clean
```
